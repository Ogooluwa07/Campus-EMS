# Campus Event Management System (React + Firebase)

## Run
1) cd client
2) npm install
3) npm run dev

## IMPORTANT: Environment Setup
Before running the project, you must set up your Firebase environment variables to connect to the database securely:
1. Create a `.env` file in the root directory.
2. Copy the contents of `.env.example` into your new `.env` file.
3. Replace the placeholder values with your actual Firebase project keys.
